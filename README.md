# Siign

Integrate [Tiime][] with [Docage][] to sign quotes.

## Configuration

You should the following environment variables:

```
# Tiime user email
TIIME_USER=
# Tiime password
TIIME_PASSWORD=
# Docage email
DOCAGE_USER=
# Docage API key
DOCAGE_API_KEY=
```

## Usage

    bundle exec puma

## Tests

    bundle exec rspec
